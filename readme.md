# dataDisplay
> 数据展示

## run
    * 方法一：
        1. `npm install`
        2. `npm start`
    * 方法二:
        1. `npm install`
        2. `npm install supervisor -g`
        3. `supervisor ./bin/www`
## 目录结构
    * bin: 执行目录
    * conf: mysql 配置文件
    * dao: mysql 交互文件
    * public: 静态文件
    * routes: 路由配置
    * util: mysql 操作工具库
    * views: 你猜
    * app.js： express主要配置文件
